# Ruby-Refine
Ruby Refine：非母语学者的英文论文表达拓宽工具
Ruby Refine 是一款专为非英语母语科研人员（Non-native Speakers）打造的轻量级桌面论文精炼沙盒。我们不鼓励华而不实的修辞，也不盲目套用期刊大白话。我们的目标是：帮你剔除拉低论文质量的“万能胶水词”，打破因表达单一导致的机械性重复，真正拓宽你的学术英语表达广度。
基于 Tauri + Rust 架构开发，纯本地运行，零内存污染，100% 保护你的未发表论文隐私。
为什么需要 Ruby Refine？
•	打破“词穷式重复”： 非母语作者在写作时，常因掌握的句型有限，在不同段落高频复读同一种句式。这不仅让论文显得稚嫩，更会导致学术查重率（Similarity Index）因局部文本高度相似而飙升。
•	反向“脱水”学术八股： 拒绝期刊中被用烂的形式主语（如 It is worth noting that...）和臃肿的名词化句式（如 conducted an investigation into）。
•	语义拓宽，直奔主题： 揪出隐藏在句子里的“学术脂肪”，引导你使用更直接的主动语态、更多元的学术强动词，让结论精准。
